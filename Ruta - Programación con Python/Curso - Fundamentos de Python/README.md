# 🚀 Sobre este proyecto

Python es uno de los lenguajes más utilizados en áreas como desarrollo backend, ciencia de datos, automatización e inteligencia artificial.

Este repositorio recopila:

- Ejercicios prácticos
- Experimentos con el lenguaje
- Ejemplos de estructuras de datos
- Prácticas de lógica de programación
- Proyecto final del curso

La idea es que funcione tanto como **bitácora de aprendizaje** como referencia futura.

# 📚 Contenidos aprendidos

Durante el curso se trabajaron los siguientes fundamentos:

## 🧠 Bases del lenguaje

- Sintaxis de Python
- Variables y convenciones
- Tipos de datos
- Conversión de tipos

## 🔁 Control de flujo

- Condicionales (`if`, `elif`, `else`)
- Operadores lógicos
- Bucles `while`
- Bucles `for`

## 🗂️ Estructuras de datos

- Listas
- Tuplas
- Conjuntos
- Diccionarios

## ⚙️ Modularización

- Funciones
- Argumentos
- Funciones lambda
- Organización del código

## ⚠️ Manejo de errores

- `try`
- `except`
- `finally`

## 📂 Manejo de archivos

- Lectura de archivos
- Escritura de archivos

# 🧩 Proyecto final

El curso culmina con un pequeño proyecto en consola que simula una **máquina de café**, integrando conceptos como:

- lógica de programación  
- control de flujo  
- estructuras de datos  
- funciones  

Este proyecto demuestra cómo pasar de **conceptos básicos a una aplicación funcional en Python**.
